# Améliorations Design - Tournoi EPS

## Changements clés (économe en code)

### 1. **Palette de couleurs épurée**
- Suppression des gradients radial complexes
- Couleurs plus nettes et contrastées
- Meilleur contraste texte/fond pour l'accessibilité
- Variables CSS optimisées

### 2. **Typographie système**
- `-apple-system, BlinkMacSystemFont, "Segoe UI"` (plus efficace que polices Google)
- Hiérarchie visuelle renforcée avec letter-spacing
- Tailles ajustées pour meilleure lisibilité

### 3. **Boutons redesignés**
- Style plus simple mais percutant
- Primary: gradient dégradé bleu épuré
- Secondary/Danger: bordure + fond inversé (design épuré)
- Ombres subtiles au hover

### 4. **Cartes de match optimisées**
- Suppression des gradients complexes
- Bordure simple + transition au hover
- Hover effect: border color + subtle shadow

### 5. **Tables et listes**
- Espacements cohérents
- Hover states simplifiés
- Colors uniformes avec les variables CSS

### 6. **Inputs et formulaires**
- Focus state avec ring shadow (élégant et accessible)
- Consistance visuelle
- Padding harmonisé

### 7. **Tabs**
- Design carré épuré (pas de pill radius)
- Transitions fluides
- States distincts (hover, active)

### 8. **Micro-détails**
- Seed pills arrondies (border-radius: 6px)
- Mini buttons cohérents
- Error text mieux visible

## Résultat
✅ Moins de code CSS
✅ Meilleur design système
✅ Plus rapide au chargement
✅ Consistance visuelle globale
✅ Accessible et professionnel
